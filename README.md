# DWAS - Smart Home Assistant

DWAS is a modern Android application designed to serve as a digital assistant for smart home management. It features a clean, professional UI with a focus on ease of use and quick controls.

## Features

- **Modern Authentication**: Secure Login and Sign-Up screens with support for Email/Password and Social Login placeholders (Google/Apple).
- **Smart Home Dashboard**: A main interface that greets the user and provides quick access to home controls.
- **AI Assistant Interface**: A built-in chat-style interface to interact with your home assistant.
- **Responsive Design**: Built using `CoordinatorLayout`, `ConstraintLayout`, and Material Design components for a fluid experience.

## Tech Stack

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: XML with [Material Components for Android](https://github.com/material-components/material-components-android)
- **Backend**: [Firebase](https://firebase.google.com/) (Authentication & Firestore)
- **Architecture**: Modern Android development practices with View Binding/Synthetics (refactoring to modern standards).

## Getting Started

### Prerequisites

- Android Studio Jellyfish or newer.
- JDK 17 or higher.
- A Firebase project (for Auth and Firestore features).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/DWAS.git
   ```
2. Open the project in Android Studio.
3. Add your `google-services.json` file to the `app/` directory.
4. Sync the project with Gradle files.
5. Run the app on an emulator or physical device.
